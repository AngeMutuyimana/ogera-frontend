# Ogera Frontend

This is the frontend for the **Ogera** application, built with **React**, **TypeScript**, and **Tailwind CSS**.


## 🧪 Setup Instructions

```bash
Node version 22   
```

### 1. Clone the project repository

```bash
git clone 'project-url'   # Replace 'project-url' with the actual repo URL
```

### 2. Move into the project directory

```bash
cd project-folder-name    # Replace with the actual folder name
```



### 3. Install frontend dependencies

```bash
npm install
```

### 5. Start the frontend server

```bash
npm run dev
```
